A library to easily create dmenu-like applications.

Dependencies
------------

- jbuilder (build)
- Base
- Stdio
- Yojson (optional)
- Cmdliner (optional, needed only to build the examples)

Building
-----------------------

    $ make

Examples
--------

In the `examples/` directory. They can be built (from the root of the repo) using:

    $ make examples
